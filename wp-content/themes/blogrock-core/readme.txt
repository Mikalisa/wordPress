== BlogRock Core Features: ==

- Theme Customizer

You can tweak a theme's settings and see a preview of those changes in real time. You'll find it in the appearance section : Appearance -> Coustomize

Theme Customizer sections:

'Site Title & Tagline' - site title is displayed on the logo area, tagline is displayed on the header area [are displayed until you add a logo and custom header]
'Logo' - the best size of image to use to upload your own logo is 222px × 45px
'Colors' - you can set: Background Color, Headers Text Color,  Sidebar Headers Color, Link Color
'Layout' - you can switch on / off fixed top bar
'Resize Components' - you can change layout width
'Breadcrumb' - you can change a separator
'Pagination' - allows you to choose between a older posts/newer posts navigation or paginated link
'Custom code' -  e.g. Google Analytics
'Background Image' - you can upload custom image which will be displayed at a resolution higher than 1280 px
'Typography options' - you can change primary font
'Navigation' - you can assign menu to an appropriate location
'Static Front Page' - display Your latest posts or a static page


- Custom Header

You can select an image to be shown at the top of your site by uploading from your computer or choosing from your media library. After selecting an image you will be able to crop it.
Images of exactly 964 × 110 pixels will be used as-is.


- Menus

BlogRock Core offers 3 menus:

'Top Pages Menu' - single-line menu at very top
'Main Menu' - multilevel main menu
'Bottom Menu' - menu in footer area

- Widgets

BlogRock Core offers 6 widgets:

'One author info' - displays short info & avatar
'Extended Recent Post' -  offers you more control when displaying recent posts
'Recent Galleries' -  displays a list of recent posts of the gallery post format, you can specify a number of posts to display.
'Recent Videos' - displays a list of recent posts of the video post format.
'Social Icons' - displays a list of social media website icons and a link to your profile.
'Video Widget' - simple widget to add video into your widget box


== Limitations: ==

Sticky post should be used with images
Photoswipe functionality works properly only with galleries which link to media file [GALLERY SETTINGS-> link to -> media file]
Custom background color or image is displayed at a resolution higher than 1280 px
